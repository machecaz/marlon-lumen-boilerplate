# Marlon Lumen Boilertemplate

## Marlon Dependencies

- [Lumen](https://github.com/laravel/lumen)
- [MongoDB](https://github.com/jenssegers/laravel-mongodb)
- [Marlon Lumen](https://github.com/machecaz/marlon-lumen)

sudo pecl install mongodb
